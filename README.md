# ✈️ AI Travel Planner

An AI-powered travel planning assistant that helps users create personalized travel itineraries, discover destinations, estimate budgets, and receive travel recommendations.

---

## 🚀 Overview

Planning a trip can be time-consuming and overwhelming. This AI Travel Planner simplifies the process by generating travel plans based on user preferences, destination choices, duration, and budget.

The system leverages Generative AI to provide smart travel recommendations and organized itineraries.

3. Provides preferences and budget
